# Math English Logic Test

Sebuah aplikasi web interaktif sederhana dan responsif untuk latihan soal matematika, bahasa Inggris, dan logika (Math, English, and Logic Test).

## Fitur Utama

- **25 Soal Latihan**: Meliputi aritmetika, persentase, rasio, pola logika, dan kemampuan pemecahan masalah dalam bahasa Inggris.
- **Timer Terintegrasi**: Batasan waktu pengerjaan selama 60 menit untuk mensimulasikan kondisi ujian sebenarnya.
- **Navigasi Soal Instan**: Memungkinkan pengguna untuk melompat ke nomor soal tertentu secara langsung.
- **Progress Bar Interaktif**: Menunjukkan persentase soal yang telah dijawab secara real-time.
- **Pembahasan Lengkap**: Setelah tes selesai, pengguna dapat melihat skor akhir beserta kunci jawaban dan penjelasan/pembahasan mendalam untuk setiap soal.
- **Desain Modern & Retro-Grid**: Antarmuka bersih yang menggunakan palet warna kontras tinggi, font Space Grotesk & IBM Plex Mono, serta visual bergaya neo-brutalism.

## Cara Penggunaan Secara Lokal

Cukup klon repositori ini atau unduh file `index.html`, lalu buka file tersebut di browser apa pun:

```bash
# Clone repository
git clone https://github.com/Emzyjeppp/math-english-logic-test.git

# Masuk ke direktori
cd math-english-logic-test

# Buka index.html (di Windows)
start index.html
```
